# Changelog

All notable changes to the Movie Booking iOS app will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-12-09

### Added
- Initial release of Movie Booking iOS app
- User authentication (Login & Sign Up)
- JWT token-based session management
- Movie browsing with carousel and grid views
- Search functionality for movies
- Category filtering (Action, Comedy, Drama, Horror, Sci-Fi)
- Interactive seat selection with visual seat map
- Date and time selection for bookings
- Real-time seat availability checking
- Booking confirmation with details
- My Bookings view to see all user bookings
- Wishlist/Favorites functionality
- User profile management
- Glassmorphic UI design with dark theme
- MVVM architecture implementation
- Combine framework for reactive programming
- Generic network layer with error handling
- Automatic JWT token injection for authenticated requests
- Image URL fixing for localhost/IP address compatibility
- Comprehensive documentation (README, API docs, Architecture guides)

### Technical Features
- SwiftUI for declarative UI
- Combine for reactive data flow
- MVVM architecture pattern
- Protocol-oriented network service
- Singleton pattern for shared managers
- Codable for JSON serialization
- AsyncImage for image loading
- UserDefaults for local persistence

### UI/UX Features
- Modern glassmorphic design
- Smooth animations and transitions
- Responsive layouts for all iPhone sizes
- Dark theme throughout the app
- Interactive components with haptic feedback
- Loading and error states
- Form validation
- Password visibility toggle
- Terms and conditions checkbox
